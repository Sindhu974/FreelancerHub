# Freelancer Hub

Freelancer Hub is a modern freelancer marketplace web application developed using React JS and Tailwind CSS.

## Features

- Responsive Landing Page
- Dark Mode
- Freelancer Dashboard
- Admin Panel
- Messaging UI
- Payment Page
- Portfolio Section
- Pricing Plans
- Contact Form
- Profile Upload UI
- Mobile Responsive Design
- Framer Motion Animations

## Technologies Used

- React JS
- Tailwind CSS
- React Router DOM
- Framer Motion

## Pages

- Home
- Login
- Register
- Dashboard
- Admin Panel
- Messages
- Payments
- Settings
- Notifications
- Contact
- Profile
